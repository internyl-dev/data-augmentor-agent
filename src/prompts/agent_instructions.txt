## MANDATORY SEARCH REQUIREMENT

**Before outputting JSON, you MUST search for ANY missing critical information:**
- Application deadlines
- Program costs/tuition 
- Eligibility requirements (grades, age, application requirements)
- Program dates (start/end dates)
- Contact information

**Search format:** "[program name] + [missing info]"
Examples: "brainyac deadline", "arise program cost", "stem program eligibility"

## GLOBAL RULES

**Output Process:**
1. Extract available information
2. Search for missing critical information
3. Provide brief search explanation
4. Output complete JSON

**Data Types:**
- **Strings:** `"not provided"` (with quotes)
- **Numbers:** `null` for age fields, `"not provided"` for other numeric fields
- **Booleans:** `true`/`false` or `"not provided"` when unclear
- **Arrays:** `[]` when empty, must be arrays even for single items

**No inference rule:** Do not guess or hallucinate information not in sources (Exception: subject field only)

## FIELD INSTRUCTIONS

### OVERVIEW
- **title:** Full program name
- **provider:** Organization running the program  
- **description:** Concise summary from program materials
- **link:** Complete URL (http/https) or "not provided"
- **subject:** Academic topics as stated OR inferred from program name/description. Use ["various"] if unclear
- **tags:** Literal keywords from content (e.g., ["free", "residential", "virtual"])

### ELIGIBILITY
- **requirements:** Use `true`/`false` for essay/recommendation/transcript, or `"not provided"`
- **other:** Array of additional requirements
- **grades:** Convert to standard format:
  - 9th→["Freshman"], 10th→["Sophomore"], 11th→["Junior"], 12th→["Senior"]  
  - Ranges: "9-12"→["Freshman","Sophomore","Junior","Senior"]
  - Non-standard: Copy exactly as written
- **age:** Use exact numbers only. For "14-18": minimum:14, maximum:18. Use `null` if not provided

### DATES
- **deadlines:** Create objects for each deadline found:
  - **priority:** "high"=final deadline, "medium"=early/priority, "low"=optional
  - **date:** mm-dd-yyyy preferred, mm-yyyy or yyyy acceptable
  - **rolling_basis:** `true`/`false`/`"not provided"`
- **dates:** Program start/end dates (same format as deadlines)
- **duration_weeks:** Numeric value only if explicitly stated (e.g., "6-week program"→"6")

### LOCATIONS
- **virtual:** 
  - `true` = online only (set state/city/address to "not provided")
  - `false` = in-person only  
  - `"hybrid"` = combines online and in-person
  - `"both available"` = choice between online or in-person versions
  - `"not provided"` = unclear
- Create separate location objects for multiple locations
- **state:** Full names only (e.g., "New York" not "NY")

### COSTS
- **free:** `true` only if explicitly free (then lowest/highest must be `null`)
- **lowest/highest:** Numeric values only, no currency symbols
- **financial_aid_available:** `true`/`false`/`"not provided"`

### CONTACT
- **email:** Complete, valid email addresses from program content
- **phone:** Complete phone numbers from program content
- Extract from main content first, then headers/footers only if program-specific
